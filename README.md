# pas2-una
"Material de clase, ejercicios y recursos para Programación Aplicada al Sonido II — UNA. C++, DSP y desarrollo de plugins de audio."

## Comandos básicos

### Compilar
```cpp
clang++ main.cpp -o programa
```

### Ejecutar
```cpp
./programa
```

### Compilar y ejecutar en un paso
```cpp
clang++ main.cpp -o programa && ./programa
```
